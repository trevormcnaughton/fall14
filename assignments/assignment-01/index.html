Assignment 1

Digital Sketches

Due September 17th, 2014

Requirements

  • 8.5 x 11 file size
  • .AI format (Illustrators native format)
  • All images in same document, with separate artboards.
  • No colors, just use black, white, grey

Description

Using the technique we discussed in class, and concepts from the Point, Line, and Plane readings. Create 9 digital sketches that express 9 different emotions, or design concepts. They can be abstract, but should only use simple shapes and lines to express.
